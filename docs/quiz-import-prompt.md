# クイズJSON生成プロンプト

このファイルには、AIにクイズのJSONデータを生成させるためのプロンプトテンプレートが含まれています。  
生成したJSONはクイズアプリの管理画面「JSONでインポート」機能から直接貼り付けてインポートできます。

---

## JSONフォーマット仕様

```json
[
  {
    "question": "問題文をここに記述する",
    "choices": [
      { "text": "選択肢A", "is_correct": false, "explanation": "この選択肢が誤りである理由（任意）" },
      { "text": "選択肢B", "is_correct": true,  "explanation": "正解の解説（任意）" },
      { "text": "選択肢C", "is_correct": false },
      { "text": "選択肢D", "is_correct": false }
    ]
  }
]
```

### フィールド説明

| フィールド | 必須 | 説明 |
|---|---|---|
| `question` | ✅ | 問題文 |
| `choices` | ✅ | 選択肢の配列（2〜4件） |
| `choices[].text` | ✅ | 選択肢のテキスト |
| `choices[].is_correct` | ✅ | `true` が正解（1つだけ） |
| `choices[].explanation` | ➖ | 補足説明（省略可） |

---

## AIへのプロンプトテンプレート

以下のプロンプトをそのままAI（ChatGPT / Claude / Gemini など）に貼り付けて使ってください。  
`【】` の部分を書き換えるだけで様々なテーマのクイズを生成できます。

---

### プロンプト（コピー用）

```
以下のJSON形式で、【テーマ: 例）HTTPの基礎知識】に関するクイズを【10】問作成してください。

## 出力形式

JSONの配列をコードブロックで出力してください。説明文や装飾は不要です。

[
  {
    "question": "問題文",
    "choices": [
      { "text": "選択肢A", "is_correct": false, "explanation": "なぜこの選択肢が誤りなのかの説明" },
      { "text": "選択肢B", "is_correct": true,  "explanation": "正解の解説" },
      { "text": "選択肢C", "is_correct": false, "explanation": "なぜこの選択肢が誤りなのかの説明" },
      { "text": "選択肢D", "is_correct": false, "explanation": "なぜこの選択肢が誤りなのかの説明" }
    ]
  }
]

## 制約条件

- 選択肢は必ず4つ用意すること
- `is_correct: true` の選択肢は各問題に必ず1つだけにすること
- 正解の選択肢の位置はランダムにすること（常にBにしないこと）
- 問題のレベルは【初級〜中級】程度にすること
- explanation（解説）は全選択肢に必ず付けること
- 正解の explanation には正しい理由・根拠を記載すること
- 不正解の explanation には「なぜその選択肢が誤りなのか」を具体的に説明すること
- 問題文は具体的かつ明確に記述すること
- 選択肢は互いに紛らわしい内容にして、理解度を問えるようにすること
```

## インポート手順

1. 上記プロンプトをAIに送信する
2. 出力されたJSONをコピーする
3. クイズアプリの管理画面（`/admin/categories/[カテゴリーID]`）を開く
4. **「JSONでインポート」** ボタンをクリックする
5. テキストエリアにJSONを貼り付ける
6. プレビューで内容を確認する
7. **「N件をインポート」** ボタンをクリックして完了

## トラブルシューティング

| エラー | 原因 | 対処法 |
|---|---|---|
| JSONの形式が正しくありません | 構文エラー | [JSONLint](https://jsonlint.com) で検証する |
| 選択肢は2〜4件必要です | 選択肢の数が範囲外 | choicesの要素数を確認する |
| is_correctはtrue/falseである必要があります | 文字列 `"true"` になっている | boolean型（クォートなし）に修正する |
| 正解（is_correct: true）の選択肢が必要です | 全選択肢がfalse | 1つをtrueに設定する |
| 正解は1つのみにしてください | 複数がtrue | trueの選択肢を1つに絞る |
